# Leap Year for Epicodus
## by Daniel Toader
### Date: April 05, 2015
#### Description
This web app lets a user check whether a given year is a leap year.

#### Setup instructions
1. Clone this git repository
2. Start the web app by loading leap-year.html in your browser
